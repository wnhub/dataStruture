一、稀疏数组和队列

1、稀疏数组

    --当一个数组中大部分是0或者相同的数时可以使用稀疏数组
    
    -------------
    行   列    值
    6    7     8 -->第一行表示数组总共有6行7列，8个不同的值
    0    3     4 -->除第一行外，其他行数据表示原始数组中第几行第几列的值是多少
    2    1     5
    ...
    -------------
